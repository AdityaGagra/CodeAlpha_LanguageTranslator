# 🌐 CodeAlpha Language Translator

A lightweight, responsive, and user-friendly web application that allows users to translate text seamlessly across multiple languages. Built entirely with vanilla frontend technologies and integrated with a translation API.

🚀 **[Live Demo](https://adityagagra.github.io/CodeAlpha_LanguageTranslator/Project1)** 

---

## ✨ Features

*   **Multi-Language Support:** Translate text across a wide variety of global languages.
*   **Intuitive UI/UX:** Clean, modern, and fully responsive layout designed for both desktop and mobile devices.
*   **Real-time Translation:** Quick processing and delivery of accurate translations.
*   **Zero Dependencies:** Built using pure HTML, CSS, and vanilla JavaScript—no heavy frameworks required.

---

## 📁 Repository Structure

The project maintains a clean, single-root architecture for easy accessibility and fast deployment:

```
├── Project1.html      # The core structure and layout of the translator web app (formerly Project1.html)
├── style.css       # Custom modern styling, layouts, and responsive media queries
├── script.js       # Handles API interaction, event listeners, and dynamic UI updates
├── LICENSE         # MIT Open Source License
└── README.md       # Project documentation and guide
```
🛠️ Tech Stack
HTML5: Semantic structure of the web application.

CSS3: Flexbox layouts, custom styling, and smooth responsive design.

JavaScript (ES6+): Asynchronous API calls, DOM manipulation, and dynamic event handling.
